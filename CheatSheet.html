World Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?

The most essential technologies which made the World Wide Web such a success is the HTTP protocol for transferring the data, the Hypertext Markup Language (=html), which made it easy to create websites and as a third the web browser as a client program, which displayed the transferred data in the described way.

2. How are resources (html pages, images, etc.) addressed in the web?

The browser resolves the server name of the URL (example.org) into an Internet Protocol address using the globally distributed Domain Name System (DNS). The browser then requests the resource by sending an HTTP request across the Internet to the computer at that address.

3. What are the components of a URL?

typical URL: https://www.example.com/index.html
http = protocol
www.example.com = hostname
index.html = file name

Web Server

1. What is a web server?

A webserver is a computer software and underlying hardware that accepts requests via HTTP or HTTPS.

2. What does a web server do?

A web server connects to the Internet and supports physical data interchange with other devices connected to the web. On the software side, a web server includes several parts that control how web users access hosted files.

3. How can content be stored on a web server?

The content on a web server is either stored on the file system or in databases.

4. How is content in webpages described?

Content is either described static or dynamic. Static content are predefined html-files, dynamic content is created while accessing the webserver (eg. as a result of a database request)

Web Client

1. What is a web client?

A web client is program which is able to retrieve data from a web server, interpreting the data and showing the data.

2. What does a web client do?

A web client is requesting or sending data from or to a web server and is working with the data.

3. Web Client and Web Browser are the same?

A web browser is specific kind of web client which shows whole website.

1. What is the html head element?

The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

2. What does !DOCTYPE html mean?

All HTML documents must start with a !DOCTYPE declaration. The declaration is not an HTML tag. It is an information to the browser about what document type to expect.

3. What are the two top most elements within the html element?

The two top most elements are paragraphs p and headers h1 - h6

4. How to emphasize and how is it displayed in the browser?

You can use the em tag to define emphasized text. The content inside is displayed in italic.

5. How to markup important text and how is it displayed in the browser?

You can markup text by using strong. The content inside is displayed bold.

Identify attributes

The a tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to The img tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed The img tag should also contain the width and height attributes, which specify the width and height of the image (in pixels) The required alt attribute for the img tag specifies an alternate text for an image, if the image for some reason cannot be displayed This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader. The style attribute is used to add styles to an element, such as color, font, size, and more You should always include the long attribute inside the html tag, to declare the language of the Web page.
